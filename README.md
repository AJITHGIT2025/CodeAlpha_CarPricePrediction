# CodeAlpha_CarPricePrediction

## Objective
Predict the selling price of a car using machine learning.

## Features Used
- Year
- Present Price
- Kilometers Driven
- Fuel Type
- Selling Type
- Transmission
- Owner Count

## Algorithm
Random Forest Regressor

## Libraries
- Pandas
- Scikit-learn
- Matplotlib

## Results
The model predicts car prices with high accuracy using historical car data.

## Dataset
Car Data Dataset
